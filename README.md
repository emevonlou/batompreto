# batompreto

Floating OCR translator (PT ↔ EN) for games, chats and on-screen text.

---

## Português

Tradutor flutuante com OCR em tempo real.

### Recursos
- Tradução PT ↔ EN
- OCR automático da tela
- Funciona em jogos, chats e qualquer texto visível
- Filtro por nicks e mensagens importantes
- Overlay leve (sempre visível)
- Copy e auto-copy

### Executar
python3 app.py

### Build
pyinstaller --onefile --windowed app.py

---

## English

Real-time floating OCR translator.

### Features
- PT ↔ EN translation
- Screen OCR detection
- Works with games, chats and on-screen text
- Smart filtering (nicks / important messages)
- Lightweight overlay (always on top)
- Copy & auto-copy

### Build
